## 🛠️ 使用方法

直接在浏览器中打开 `index.html` 文件即可浏览网站。

```bash
# 或者使用本地服务器
# Python 3
python -m http.server 8000

# Python 2
python -m SimpleHTTPServer 8000
```

然后访问 http://localhost:8000
